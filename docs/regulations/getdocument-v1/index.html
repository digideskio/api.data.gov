Document Information

This API allows users to request metadata associated with the specific Document.

    Request URL

    GET http://api.data.gov/regulations/getdocument/v1.format?parameters

    Sample Requests

    1. Using FR Number: 
      http://api.data.gov/regulations/getdocument/v1.xml?api_key=DEMO_KEY&FR=2011-31912
    2. Using Document ID: 
      http://api.data.gov/regulations/getdocument/v1.xml?api_key=DEMO_KEY&D=EPA_FRDOC_0001-10857
    3. Using JSON output: 
      http://api.data.gov/regulations/getdocument/v1.json?api_key=DEMO_KEY&FR=2011-31912

    Request Parameters

    Parameter Required Argument Value
    API Key Yes api_key Enter your API Key (an alphanumeric key provided upon request)
    One of the following must be present:
    Document ID No D FDMS Document ID
    FR Number No FR Document FR Number
