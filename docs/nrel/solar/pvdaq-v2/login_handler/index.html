Login (Deprecated) (POST /api/georeserv/login_handler)

Deprecated: We encourage you to migrate to version 3 of the PVDAQ API.

Login to utilize access-controlled Georeserv services.

For login credentials, contact Ryan Smith. Examples on how to use this service can be found in the demo project.

    Request URL

    GET /api/georesrv/login_handler?api_key=DEMO_KEY

    Request Parameters

    Parameter Required Value Description
    api_key Yes
    Type: string
    Default: None

    Your developer API key. See API keys for more information. (URL Param)
    login Yes
    Type: string
    Default: None

    Username to login with. (POST Param)
    password Yes
    Type: string
    Default: None

    Password to login with. (POST Param)

    Response

    If the login is successful, a HTTP 302 redirect will be issued. Within the heders will be an authentication token which can be used in subsequent requests. (Also known as a cookie).

    Rate Limits

    Standard rate limits apply. No more than 1,000 requests may be made in any hour. No more than 10,000 requests may be made in any day.

    Errors

    Standard errors may be returned. In addition, the following service-specific errors may be returned:

    HTTP Status Code Description
    401 Unauthorized. The server was unable to validate your credentials.
