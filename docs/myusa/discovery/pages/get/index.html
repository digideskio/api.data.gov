Get a page by ID

Retrieve a page by its ID

    Request URL

    http://api.data.gov/myusa/discovery/pages/:id?parameters

    Request Parameters

    Parameter Required Value Description
    api_key Yes
    Type: string
    Default: None

    Your developer API key. See API key usage for more information.
    id Yes
    Type: integer
    Default: None
    		The Page ID
    related No
    Type: integer
    Default: None
    		Number of related pages to return, must be between 0 and 25.
    callback No
    Type: string
    Default: None
    		Optional JSONP callback with which to wrap the response

    Rate Limits

    Standard rate limits apply. No more than 1,000 requests may be made in any hour. No more than 10,000 requests may be made in any day.

    Errors

    Standard errors may be returned.
